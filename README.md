
# PortMaster DZ ⚓

نظام إدارة الميناء الذكي للتحقق من هويات الصيادين عبر تقنية NFC وتسجيل الإنتاج اليومي.

## المميزات الرئيسية
- **تحقق NFC**: قراءة بطاقات الصيادين والتأكد من صلاحية التأمين.
- **إدارة الصيادين**: إضافة وتعديل بيانات الصيادين (ID، السفينة، التأمين).
- **سجل الصيد**: تسجيل الكميات اليومية وأنواع الأسماك.
- **تقارير**: تصدير تقارير شهرية بصيغة PDF و Excel.

## المتطلبات

- Node.js (v18 أو أعلى)
- npm أو yarn

## التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. إنشاء ملف البيئة
انسخ `.env.example` إلى `.env.local`:
```bash
cp .env.example .env.local
```

تأكد من تعديل `VITE_API_URL` بحيث يشير إلى عنوان خادمك:
```
VITE_API_URL=http://localhost:5000
```

### 3. التطوير والتشغيل
```bash
npm run dev
```

سيفتح التطبيق على `http://localhost:3000`

### 4. الترجمة إلى الإنتاج
```bash
npm run build
```

سيتم إنشاء مجلد `dist/` يحتوي على الملفات الجاهزة للنشر.

## المعاينة السريعة (وضع المحاكاة)

عند عدم توفر خادم API:
- **بيانات المستخدم الافتراضية**:
  - البريد: `admin@port.com` - كلمة المرور: `123456`
  - البريد: `nfc@port.com` - كلمة المرور: `123456`
  - البريد: `csns@port.com` - كلمة المرور: `123456`

- البيانات في هذا الوضع تُحفظ في الذاكرة فقط

## البنية التحتية المطلوبة

يتطلب التطبيق خادماً في الخلفية يدعم الـ API التالية:

### الـ Endpoints المطلوبة
- `POST /auth/login` - تسجيل الدخول
- `GET /fishers` - الحصول على قائمة الصيادين
- `POST /fishers` - إضافة صياد جديد
- `PUT /fishers/:id` - تحديث بيانات الصياد
- `DELETE /fishers/:id` - حذف صياد
- `POST /nfc/scan` - محاكاة ماسح NFC
- `POST /nfc/logs` - حفظ سجل الفحص
- `GET /nfc/logs` - الحصول على سجلات الفحص
- `GET /catches` - الحصول على سجل الصيد
- `POST /catches` - إضافة سجل صيد جديد
- `PUT /catches/:id` - تحديث سجل الصيد
- `GET /renewals` - الحصول على سجلات التجديد
- `POST /renewals` - إنشاء تجديد جديد
- `POST /reports/upload` - رفع ملف التقرير
